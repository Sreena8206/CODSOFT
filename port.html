Projects

Multimodal AI Chatbot with Inclusive Communication

Integrating Sign Language Input

This project develops an inclusive communication app for individuals with hearing and speaking impairments, featuring an AI chatbot that utilizes Natural Language Processing (NLP) and Computer Vision. The chatbot can interpret text-based inputs and sign language gestures, leveraging OpenAI for language processing and OpenCV for gesture recognition. It also includes speech-to-text capabilities to accommodate diverse user needs. The goal is to create a user-friendly, accessible interface that bridges the gap between sign language and technology, promoting inclusivity in digital interactions.

Video Conferencing Site

With Real-Time Translation & Summarization

This project aims to develop an advanced online meeting platform featuring real-time translation and selective summarization. All participants' speech will be translated into multiple languages, ensuring seamless multilingual communication. Meeting organizers can select specific participants whose speech will be stored and summarized using advanced NLP techniques. The platform will incorporate secure user authentication, email verification, and notifications, enhancing user experience and data security. Key technologies include WebRTC for real-time communication, cloud services for speech recognition and translation, and machine learning models for summarization. The project will be implemented in phases, ensuring thorough testing, optimization, and scalability.
